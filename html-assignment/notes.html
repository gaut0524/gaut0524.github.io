HTML tags and description

  • The <head> tag is a container for metadata (information like title, author, creation date and more) and is placed between the <html> tag and the <body> tag.
  • The <paragraph> tag is a paragraph tag used for structure and organization. I used this tag on my Home Page.
  • The <ul> tag is used for unordered lists, where torder of appearance does not matter.
